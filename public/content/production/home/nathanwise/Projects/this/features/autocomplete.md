This site provides the typical autocomplete to significantly improve QoL when using the terminal. 

This site has implemented the following:
- Path autocomplete for Files & Directories
- Command name autocomplete
- Command specific argument/flag autocomplete

To use autocomplete press the `Tab` key. Try the following...
1. Type `cat ~/Projects/` in the terminal
2. Press `Tab` to see suggestions
3. Continue typing so that the input is `cat ~/Projects/t`
4. Press `Tab` to autocomplete to `cat ~/Projects/this`
