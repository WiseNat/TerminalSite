Similar to version 1.0, you can switch the current (Shell) Flavour and Theme.

The Shell Flavour itself is merely a visual text change for the initial prompt and PS1 prompt. 
Try `terminal -f Windows` and `reboot` to see both.

The Theme is a visual change for the colours and font used in the terminal.
Try `neofetch` and `terminal -t Classic` to see this clearly.

For a list of available Flavours & Themes, run `terminal -L`.
