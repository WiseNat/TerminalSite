Welcome to my portfolio site - everything on the site is accessible via Linux-like commands.

If you're unfamiliar with terminals and just want to see my CV, then just type 'download ~/Documents/CV.pdf' and press Return/Enter to download my CV.
I'd also recommend viewing this site on Desktop, a few of the QoL features are difficult to use on mobile due to mobile keyboard layouts.

# Useful Commands
- 'help' to see a list of commands, and 'help <commandName>' (e.g. 'help cat') to see individual command help pages
- 'ls' to view the contents of the directory you're currently in
- 'cat <filePath>' (e.g. 'cat /etc/os-release') to read a file
- 'download <filePath>' (e.g. 'download ~/Documents/CV.pdf') to download a file
- 'cd' to change your current working directory
- 'pwd' to view your current working directory
- 'tree <filePath>' (e.g. 'tree / --prune -a') to view recursively child directories & files

# Features
- Command History (using Arrow Keys)
- Autocomplete (using Tab)
- Realistic Command & Argument Parsing
- Terminal Flavours & Themes
- File & Directory Navigation

Run 'cat ~/help.md' at any time to view this again!
