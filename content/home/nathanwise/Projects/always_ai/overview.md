In 2020, for 3 months, I was the Lead Developer for a Discord bot for [alwaysAI](https://alwaysai.co/).
This bot allowed users to run alwaysAI's deep vision models from within Discord.
It also provided features for searching their documentation and viewing model information.

A [blog post](https://web.archive.org/web/20211019112450/https://alwaysai.co/blog/computer-vision-alwaysai-discord-bot-creators) was created by alwaysAI, which has been archived.

This was the first programming project I worked on with another person.
