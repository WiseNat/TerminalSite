When I was working on a Tkinter project, I conceptualised and developed the Stylehandler library.
This allows for the use of re-usable styles that can be applied to any TK widget.

It was created due to flaws with the TTK style system, see [here](https://stackoverflow.com/questions/17635905/ttk-entry-background-colour).

This was used in one of my University projects to provide consistent styling across the application.
An example of how to use it is available [here](https://github.com/WiseNat/StyleHandler/blob/main/samples/HoverButton.py).
